# CPython Grammar Rule Inventory

This file tracks every rule currently present in `/Volumes/samsung/GitHub/cpython/Grammar/python.gram`.
It is intentionally broader than `tests/cpython_coverage.md`: rows marked `missing` are grammar rules that MiniPython has not yet tracked as explicit coverage rows.

Generated from the local CPython checkout and current MiniPython coverage table.

This inventory measures grammar-rule coverage only. A `supported` row means
MiniPython has explicit parser/tokenizer coverage for that CPython grammar
production or its error-production analogue. It does not mean the full CPython
runtime semantics, object model, import system, standard library behavior, or
the corresponding `Lib/test` module has been fully migrated.
